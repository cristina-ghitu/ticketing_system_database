The purpose of this project is to practice designing and managing relational databases in a realistic business scenario. It is suitable for learning SQL fundamentals and data management techniques used in real IT support systems.

Technologies Used
- SQL
- MySQL relational database management system



Database Structure

The project currently includes the following tables:
| Table      | Purpose                            |
| ---------- | ---------------------------------- |
| `users`    | Stores user accounts and roles     |
| `tickets`  | Stores support tickets             |
| `comments` | Stores ticket discussions/comments |

Relationships between tables are enforced using foreign keys to maintain data integrity.
